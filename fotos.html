Materiales y Herramientas Utilizados

Cajetin Rack

Es un armario con base y estructura metálica diseñado para alojar todos los sistemas informáticos y de telecomunicaciones que necesitan las empresas u organizaciones donde están instalados. Estos armarios son utilizados para almacenar de forma segura servidores, switches, routers profesionales, sistemas de almacenamiento, sistemas de alimentación ininterrumpida (SAI) y todo el cableado de red necesario para proporcionar los diferentes servicios a la red local.

24-Port Gigabit Switch

Un switch de 24 puertos es un dispositivo de red que permite conectar múltiples dispositivos, como computadoras, impresoras, servidores, y otros equipos, a una red local. Estos switches ofrecen la capacidad de gestionar y dirigir el tráfico de datos entre los dispositivos conectados, lo que facilita la comunicación y el intercambio de información en una red empresarial o doméstica.

Cable UTP

Significa Unshielded Twisted Pair (par trenzado no apantallado), es un tipo de cable de red que consta de 8 hilos divididos en cuatro pares trenzados, con colores azul, verde, naranja y marrón. Este tipo de cable es comúnmente utilizado para conexiones a Internet y forma parte de la instalación interior de las conexiones a Internet.

Conector RJ-45

El conector RJ-45 se utiliza para conectar dispositivos de red, como computadoras, impresoras y switches, a una red Ethernet. Es un conector de 8 posiciones y 8 contactos (8P8C) que se utiliza para cables de red Ethernet.

Coupler Rj45 Hembra

Se utiliza para extender la longitud de los cables de red conectando dos cables macho RJ45 juntos. Son ideales para situaciones en las que se necesita extender la longitud de un cable de red sin la necesidad de reemplazarlo por uno más largo.

Bastidor de 3 módulos

Es una estructura básica que permite la instalación de hasta tres elementos eléctricos o electrónicos, como interruptores, enchufes o reguladores de luz.

Cajetin Superficial Para Red LAN

Es utilizado en instalaciones de redes estructuradas para acomodar y organizar cabos de red.

Canaletas para cables PVC

Son conductos especializados que se utilizan para guiar cables y alambres eléctricos.

Taladro

Es una herramienta eléctrica o manual utilizada para perforar superficies, como madera, metal, plástico y albañilería.

Crimpadora de red LAN

Es una herramienta utilizada para conectar conectores RJ45 a cables de red Ethernet. Estas herramientas son esenciales para la instalación y mantenimiento de redes de área local (LAN) y se utilizan para asegurar los conectores a los cables de red de manera segura y confiable.
